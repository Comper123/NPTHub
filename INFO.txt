Создание ssh ключа
ssh-keygen -t rsa

Вывод в консоль публичного ssh ключа
cat ~/.ssh/id_rsa.pub

Кидаем публичный ssh в GitHub
При вводе чтобы не было в поле перевода на другую строку

Клонирование
git clone <Ссылка на репозиторий>.git

Добавление репозитория
git remote add <Название для себя> <Ссылка на репозиторий>.git

Связываем локальную ветку с серверной во избежание проблемы
git merge --allow-unrelated-histories <имя локального репозитория>/<ветка на github>

Внесение изменений на сервер
git push <репозиторий у себя> <название ветки>

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Подключенные репозитории
git remote

История commit'ов
git log

Запушить репозиторий с папки заходим в которо проект и:
git push <имя подключенного репозитория> <имя локальной ветки которую пушим>

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Как сохранить изменения в удаленный репозиторий:

1. Добавляем все файлы
git add .

2. Делаем локальный коммит (все файлы перестанут подсвечиваться)
git commit -m "<Подпись к коммиту (кратко что сделали)>"

3. Сливаем ветку в репозиторий
git push <имя репозитория локального (remote)> <имя ветки в которую пушим>
